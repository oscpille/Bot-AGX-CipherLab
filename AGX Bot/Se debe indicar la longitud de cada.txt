Se debe indicar la longitud de cada dato a capturar según la base de datos del cliente.

Abierto: Ese no tiene "candado", lo que se escanea pasa, no se le carga una base de datos o catálogo.

Cerrado: Se agrega una base de datos, se le pone un candado cuando x código no está dentro de la base no se toma en cuenta, se muestra un código de no encontrado.

Forzado: Se le carga una base pero también deja entrar códigos que no están dentro de la base.

Como nota, en todos es necesario escanear con el scaner y tecleando, en caso de que el lector no quiera leer un código este se digita manualmente.
